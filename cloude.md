BlueCore is a modular browser architecture designed for scalability and performance.

## Core Pillars
1. **Lightweight Start**: Defaulting to Tauri/WebView for speed.
2. **Power on Demand**: Opt-in Chromium engine for full compatibility.
3. **Modular First**: AI and Privacy features built as decoupled modules.
